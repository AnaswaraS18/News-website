# Global News Network 🌍📰

A responsive, multi-page beginner-friendly news website built to practice fundamental web development skills. Experience a structured web application that demonstrates core HTML, CSS, and JavaScript techniques.

## ✨ Features

- **Multi-Page Layout**: Contains essential web pages including Home, Latest News, and Contact Us.
- **Responsive Design**: Designed with modern UI/UX principles to ensure a seamless experience across desktop and mobile.
- **Interactive Elements**: Features basic JavaScript interactions such as dynamic greeting alerts.
- **Contact Form Validation**: Form validation built into the Contact Us page for interacting with users.
- **Clean Codebase**: Fully commented, beginner-friendly raw HTML, CSS, and JS files, perfectly structured for learning and scaling.

## 🚀 Technologies Used

- **HTML5**: For crafting the structural markup and semantic elements.
- **CSS3**: For advanced styling, layout structuring, and creating an engaging visual experience.
- **Vanilla JavaScript**: To power user interaction and simple dynamic events across the website.

## 🛠️ Project Structure

```text
├── images/
│   └── breaking_news.png    # Image assets used across the site
├── index.html               # The main landing page
├── news.html                # Displays the latest stories and reports
├── contact.html             # A contact form for user inquiries
├── style.css                # Global stylesheet ensuring visual consistency
└── script.js                # Core logic for alerts and validation
```

## 💻 Getting Started

Follow these instructions to run the project locally on your machine.

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/global-news-network.git
   ```

2. **Navigate to the directory**:
   ```bash
   cd global-news-network
   ```

3. **Open the project**:
   - Simply double-click `index.html` to open it in your default web browser.
   - Alternatively, open the project folder in your preferred code editor (such as VS Code) and use a local development server like the "Live Server" extension for a better development experience.
