🌍 Global News Network 📰

A responsive, multi-page beginner-friendly news website built to practice fundamental web development skills. This project demonstrates core concepts of HTML, CSS, and JavaScript through a structured and interactive web application.

«⚡ This project was initially built with the help of Antigravity and further customized to better understand frontend development concepts.»

---

✨ Features

- Multi-Page Layout: Includes essential pages like Home, Latest News, and Contact Us
- Responsive Design: Optimized for both desktop and mobile devices
- Interactive Elements: Basic JavaScript features like greeting alerts
- Contact Form Validation: Ensures user input is handled correctly
- Clean Codebase: Well-structured and commented files for easy understanding and scalability

---

🚀 Technologies Used

- HTML5 – Semantic structure and content
- CSS3 – Styling, layout, and responsiveness
- Vanilla JavaScript – Interactivity and dynamic behavior

💻 Getting Started

Follow these steps to run the project locally:

1. Clone the repository
   
   git clone https://github.com/AnaswaraS18/global-news-network.git

2. Navigate to the project folder
   
   cd global-news-network

3. Run the project
   
   - Open "index.html" in your browser
   - OR use VS Code + Live Server extension for better experience
